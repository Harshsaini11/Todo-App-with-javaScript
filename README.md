This code is a simple command-line To-Do List application in JavaScript. It allows users to manage tasks by entering commands such as add, list, delete, and quit. The program uses an array (todo) to store tasks and continuously runs inside a while(true) loop until the user chooses to exit.

The application works as follows:

Add: Adds a new task to the to-do list.
List: Displays all stored tasks with their index numbers.
Delete: Removes a task from the list using its index number.
Quit: Stops the application and exits the loop.

The program uses prompt() for user input and console.log() for displaying output, making it a basic interactive task management system.

Open the HTML file in Visual Studio Code (VS Code).
Run the HTML file using Live Server or open it directly in the browser.
After the webpage opens in the browser, right-click anywhere on the page.
